Learning Git & GitHub

Cashar aan ku baranayo sida loo isticmaalo Git, GitHub, iyo GitHub Pages

Waa Maxay Git?

Git waa software kaa caawinaya in aad la socoto isbeddelada aad ku sameyso code-ka (Version Control System). Waxaa uu kuu oggolaanayaa:

  • Inaad kaydiso noocyo kala duwan oo project-kaaga ah (commits)
  • Inaad dib ugu noqoto version hore haddii wax khaldan yihiin
  • Inaad si fudud ula shaqeyso dadka kale

Amarrada Muhiimka ah ee Git

Command Sharaxaad
git init Wuxuu sameeyaa repository cusub (ku bilaabaa Git folder-ka).
git add . Waxay diiwaangelisaa dhammaan faylasha cusub ee la beddelay si loo commit-gareeyo.
git commit -m "message" Waxay kaydisaa isbeddelada adigoo ku lifaaqaya fariin (message).
git branch -M main Waxay magacaabeysaa laanta ugu weyn ee mashruuca "main".
git remote add origin <URL> Waxay ku xireysaa mashruucaaga GitHub repository-ga.
git push -u origin main Waxay code-ka u diraa GitHub repository-ga si uu online u galo.

Waa Maxay GitHub?

GitHub waa website kuu oggolaanaya inaad kaydiso code-ka Git-ka, la wadaagto, oo aad ku sameyso project-yadaada si online ah.

  • Waxaad abuurtaa repository si aad ugu kaydiso code-ka
  • Waxaad isticmaashaa git push si aad u dirto code-ka GitHub
  • Waxaad arki kartaa code-kaaga meel kasta oo internet ah

GitHub Pages - Daabacaadda Website-ka

GitHub Pages waxay kuu oggolaanaysaa inaad website-kaaga si bilaash ah u daabacdo. Marka aad sameyso repository, waxaad tagtaa:

  1. Settings → Pages
  2. Dooro "Deploy from a branch"
  3. Branch = main
  4. Riix Save

Kadib dhowr daqiiqo, website-kaaga wuxuu toos uga furmayaa: https://username.github.io/repository/

Faa’iidooyinka Git & GitHub

  • Kaydin iyo ilaalin code
  • La shaqeyn fudud oo kooxeed
  • La socod isbeddelo (version history)
  • Hosting bilaash ah (GitHub Pages)
  • Inaad si xirfad leh u dhisto portfolio
